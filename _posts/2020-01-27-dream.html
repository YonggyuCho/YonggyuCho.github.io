1. jest 설치 및 설정

jest 설치: npm install jest --save-dev

설치후 package.json 파일 안의 다음 부분을 수정합니다 

        
{
  "scripts": {
    "test": "jest"
  }
}

* 테스트 실행: npm run test

특정 테스트만 실행 : npm run test <파일명 or 경로>

2. jest에 사용되는 함수

test() 함수

가장 일반적으로 쓰이는 test()이다.

        
test("테스트 설명", () => {
  expect("검증 대상").toXxx("기대 결과");
});
describe() 함수

describe안에는 test함수가 들어가야한다

describe는 test함수들을 묶는 그룹이라 생각하면 된다.

toXXX (Test Matcher) 종류

- toBe() : 같다 - toEqual() : 같다 toBe는 객체인 것들은 안된다. 그럴 때 쓰는 게 Equal Jest는 toBe() 대신에 toEqual() 함수를 사용하라고 가이드해주고 있습니다 - toBeGreaterThanOrEqual() : 이상 - toBeLessThanOrEqual() : 이하 - toBeNull() : Null이 되는 경우를 검사한다. - toBeTruthy() : True인지 검사 - toBeTruthy(), toBeFalsy() - toHaveLength(), toContain() - toMatch() - toThrow()
